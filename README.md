# raw-http-server

A raw HTTP server built from scratch in Python, using only sockets.

## What it does

Listens on a port, receives a GET request and responds with a simple HTML page.

## Why I built this

To understand what happens between the browser and the server at a low level — without frameworks hiding the details.

## Tech

- Python 3
- socket (standard library only)
- Linux / Ubuntu
